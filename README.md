# IBM Applied Data Science Capstone
## SpaceX Falcon 9 First Stage Landing Prediction
### Project Overview
This repository contains my final capstone project completed as part of the **IBM Applied Data Science Professional Certificate**.
The objective of this project is to predict whether the **Falcon 9 first stage** will land successfully after launch. Successful landings allow SpaceX to reuse boosters and significantly reduce launch costs.

## Project Objectives
* Collect SpaceX launch data from the SpaceX REST API.
* Perform web scraping using Wikipedia.
* Clean and prepare the data.
* Perform exploratory data analysis (EDA).
* Query the data using SQL.
* Create interactive maps with Folium.
* Build an interactive dashboard using Plotly Dash.
* Develop machine learning models to predict Falcon 9 first-stage landing success.

## Project Workflow

1. Data Collection
2. Data Wrangling
3. Exploratory Data Analysis
4. SQL Analysis
5. Interactive Visual Analytics
6. Folium Maps
7. Plotly Dash Dashboard
8. Machine Learning Models
9. Final Presentation
